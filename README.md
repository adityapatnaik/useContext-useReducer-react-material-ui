# A Simple Demonstration of ReduxStyle-State-Management in React using useReducer,useContext and Material-UI: Todo App



## Setup
```
npm i
```

## Get Started

```
cd src
npm start
```

## How to use?

- Click on Add to add a new Todo
- Tap to delete
